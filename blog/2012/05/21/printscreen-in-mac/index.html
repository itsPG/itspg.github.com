[Mac] 關於Mac的螢幕截圖 (Printscreen)

May 21st, 2012

這篇筆記了一下怎樣在Mac裡面截圖，以及在mac bootcamp的windows下，怎樣擷取windows下的圖片

1. On Mac

Shift + Command + 3

擷取整個畫面

Shift + Command + 4

利用滑鼠圈選擷取範圍

Shift + Command + Space + 4

擷取指定視窗

2. On Windows

Fn + Shift + F11

擷取整個營幕畫面(等同於printscreen)

Fn + Shift + Alt + F11

可以擷取正在使用中的視窗畫面(等同於Alt + printscreen)
